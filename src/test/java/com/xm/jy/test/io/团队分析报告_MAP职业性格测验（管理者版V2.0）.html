阅读指引

测验介绍
MAP职业性格测验是一套适合中国文化背景、应用于职业环境、全面系统的性格测验。它以智鼎公司经典的胜任力MAP模型为理论基础，根据严格规范的程序进行开发和修订，经大量施测和应用，具备良好的信度和效度指标。
测验从M（思维倾向）、A（态度和动力）、P（人际特点）3个方面测评性格特征：
M：创新、前瞻性、理论思考、质疑、数据导向、实践性、关注细节
A：情绪稳定、乐观性、坚韧性、尽责、守规性、主动性、成就动机、好胜心、内省性、果断性
P：社交自信、乐群性、人际敏锐、助人倾向、支配性、说服他人、自主性
通过全面、细致地描绘团队成员的性格轮廓，测验可根据不同岗位、不同需求和不同用途将各性格维度进行灵活组合，以对团队的胜任力潜质、情商、领导风格、决策风格、职业倾向、团队角色等特征进行有效的预测。从而了解团队成员表现突出之处，以及相对不足之处，可用于团队建设、团队发展和培训需求分析。
特别说明
  • 1. 本测评报告基于个体对自己的评价得出，结果准确性受其当时的心理状态、作答认真和坦诚程度等因素影响，请合理参考和使用；
  • 2. 本测评报告中的性格维度有特定的含义，请根据性格轮廓报告中的内容来准确理解，而不要以工作生活经验进行简单推断，以免造成报告的误解和误用；
  • 3. 团队报告主要用于了解团队整体情况，而不等于团队中每个成员都具备团队整体的特点，如果想了解某成员的情况，需结合其个人报告和具体情况进行分析；
  • 4. 团队人员、团队领导的变动，团队目标的变更等，都会影响报告的有效性，建议本报告的有效期为12个月。
报告有效性
作答有效性用于确定测评结果是否有效，主要包括作答完成率、作答时间、选项分布三个指标。这三个指标的判定标准为：
  • 1. 作答完成率：测验题目完成率≥88%；
  • 2. 作答时间：测验作答时间≥12分钟；
  • 3. 选项分布：所有选项选择率＜60%，表示作答者没有使用某种固定的模式作答。
若有一个指标没有达到标准，本报告呈现的测评结果将被视为无效。无效的报告参考价值不大，表明作答者在答题过程中不太认真。

领导风格

领导风格指的是个体在长期的个人经历或领导实践中逐步形成的行为方式。本报告中领导风格分为以下五种类型：
  • 指挥型：高指导-低支持，指导下属如何完成任务，并对完成任务的时间和目标予以具体的指令。
  • 影响型：高指导-高支持，给下属明确的任务指令，同时也会鼓励下属积极行动，完成任务。
  • 支持型：低指导-高支持，和下属共同面对问题，给予支持和鼓励，维护民主的团队氛围。
  • 授权型：低指导-低支持，将任务交由下属独立负责，给下属较大的自行支配的空间。
  • 均衡型：一般会给予下属一定的指导，并提供适当的资源支持。
