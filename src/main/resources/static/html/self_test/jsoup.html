阅读指引

测验介绍
校园招聘测评方案从能力和性格两个方面考察候选人，帮助企业在大规模群体中快速筛选出优秀人才。综合建议是综合了思维能力和心理健康指数两类信息后给出的，当任何一方面低于标准时，我们都建议您慎重选择。
思维能力可以预测候选人在未来工作中的上手速度和工作业绩，用来淘汰不够“聪明”的人。可以作为识别优秀人才的门槛能力。
心理健康指数是通过性格来预测的。当心理健康指数位于警戒线以下时，表明候选人未来可能会表现出影响工作绩效的行为，例如过于悲观、不善交际、难以适应变化、逃避责任等，存在一定的任用风险。
此外报告还提供了胜任潜质特征、人岗匹配度、职业性格特征等方面的信息，有助于企业从更多的角度了解目标团队。需要注意的是：这部分的结果是通过性格预测的，不建议用于排名或筛选。

为了快速而直观地筛选出优秀人才，报告中将候选人的作答结果分为不同的等级，具体的等级说明如下：
当心理健康指数≥48分时，心理健康状况正常；当心理健康指数<48分时，其分数低于常模群体2%的人，心理健康状况值得关注。胜任潜质得分等级为：1-3分为有待提高；4-7分为潜力中等；8-10分为潜力较高。

特别说明
  • 1.本测评报告结果的准确性受个体当时的心理状态、作答认真和坦诚程度等因素影响，请合理参考和使用，建议本报告的有效期为12个月，如需帮助请邮件至product@zhiding.com.cn；
  • 2.本测评报告中的维度有特定的含义，请根据报告中的内容来准确理解，而不要以工作生活经验进行简单推断，以免造成报告的误解和误用；
  • 3.团队报告主要用于了解团队整体情况，而不等于团队中每个成员都具备团队整体的特点，如果想了解某成员的情况，需结合其个人报告和具体情况进行分析。
作答有效性指标
阅读本报告时，请先关注团队成员整体的作答有效性情况。思维能力测验与性格测验的作答有效性共同决定本方案测评结果是否有效，若其中任何一部分测验作答无效，个人报告呈现的测评结果将被视为无效。无效的报告参考价值不大，表明作答者在答题过程中不太认真。

领导风格

领导风格指的是个体在长期的个人经历或领导实践中逐步形成的行为方式。本报告中领导风格分为以下五种类型：
  • 指挥型：高指导-低支持，指导下属如何完成任务，并对完成任务的时间和目标予以具体的指令。
  • 影响型：高指导-高支持，给下属明确的任务指令，同时也会鼓励下属积极行动，完成任务。
  • 支持型：低指导-高支持，和下属共同面对问题，给予支持和鼓励，维护民主的团队氛围。
  • 授权型：低指导-低支持，将任务交由下属独立负责，给下属较大的自行支配的空间。
  • 均衡型：一般会给予下属一定的指导，并提供适当的资源支持。
